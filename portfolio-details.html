This is a responsive homepage clone inspired by famous websites, built using HTML, CSS, and Tailwind CSS. It’s currently in the early stages of development and focuses solely on the frontend structure — with no JavaScript or backend functionality included yet. The main goal of this project is to showcase clean layout design, responsive styling, and modern UI principles using utility-first CSS via Tailwind. It reflects my understanding of structuring a real-world landing page that is both visually appealing and scalable.
